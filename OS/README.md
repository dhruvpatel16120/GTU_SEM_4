## 📖 Official Syllabus Units (BE04000221)

> **TIP**
> **Units 2, 4, and 6** carry the highest marks and calculation-based problems!

| Unit             | Title & Major Topics                                                                                            | Weightage |
| ---------------- | --------------------------------------------------------------------------------------------------------------- | --------- |
| **Unit 1** | **Introduction:** OS Goals, Architecture, System calls, Batch/Multi-programming modes.                    | 15%       |
| **Unit 2** | **Proc. & Thread Management:** Scheduling algorithms (FCFS, SJF, RR), `fork()`, `exec()`, `join()`. | 12%       |
| **Unit 3** | **Concurrency & Sync:** Mutual Exclusion, Semaphores, Pipes, Signals, Monitors.                           | 12%       |
| **Unit 4** | **Inter Process Comm (IPC):** Peterson’s Solution, Producer-Consumer, Reader’s-Writer Problem.          | 14%       |
| **Unit 5** | **Deadlock:** Prevention, Avoidance (Banker's Algorithm), Detection, Starvation.                          | 7%        |
| **Unit 6** | **Memory Management:** Paging, Allocation (Best/First/Worst Fit), VM replacement (LRU, FIFO, Optimal).    | 16%       |
| **Unit 7** | **I/O & Disk Scheduling:** FCFS, SCAN, SSTF, RAID, Disk Cache management.                                 | 11%       |
| **Unit 8** | **Unix/Linux OS:** Shell Scripting, Shell Filters, System Administration.                                 | 11%       |
| **Unit 9** | **Virtualization:** VMware ESXi, Hyper-V, Xen Variants, Android VM Architecture.                          | 12%       |

## 🎓 Exam Writing Tips

- **For scheduling questions:** Always draw Gantt chart first
- **For synchronization:** Show semaphore operations clearly
- **For deadlock:** Use resource graphs or matrices
- **For paging:** Show page tables and address translation
- **For calculations:** Show all steps, not just final answer
- **For comparisons:** Use table format for clarity

## 📖 Study Material Reading Order

**Optimal sequence:**

1. Read `OS_roadmap.docx` - Get a bird's eye view of the entire course.
2. Read `Unit-1.docx` to `Unit-4.docx` - focus on process lifecycles and IPC logic.

## 🛠️ Practical Topics

Major practical challenges mentioned in the syllabus:

- **Practical 1-3:** Basic and advanced Linux shell commands, Shell scripting logic.
- **Practical 4:** System calls: Creating `n` child processes using `fork()`.
- **Practical 5:** Thread parallelism vs Process parallelism (Arithmetic calculations).
- **Practical 6:** Shared memory variables and Semaphores in Linux.
- **Practical 7:** Custom `cd` command using shell logic.
- **Practical 8:** Displaying 5 largest files and user management scripts.
- **Practical 10:** File extension manipulation and content translation.

### Study Tips:

- 📌 **Visual learning:** Draw Gantt charts for CPU scheduling and page replacement tables.
- 📌 **Hands-on practice:** Test your shell scripts in a Linux terminal or WSL.
- 📌 **Banker's Algorithm:** Practice this multiple times – it's a high-frequency exam problem.
- 📌 **Paging:** Understand how logical addresses map to physical ones.
- 📌 **Time management:** Focus on Unit 6 (Memory) early as it's the most mathematically detailed.
- 📌 Focus on **2024-2025** papers for the latest GTU question patterns.
