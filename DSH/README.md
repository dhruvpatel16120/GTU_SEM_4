## 📖 Official Syllabus Units (BE04000281)

> **TIP**
> Focus on **Units 3, 4, and 7** for the highest scoring potential as they cover probability and classification.

| Unit       | Title & Major Topics                                                                                         | Weightage        |
| ---------- | ------------------------------------------------------------------------------------------------------------ | ---------------- |
| **Unit 1** | **Introduction to Data Science:** Data engineering, Analytics vs Reporting, Big Data, ML Frameworks.         | 5% (Suggestive)  |
| **Unit 2** | **Descriptive Analytics:** Data Types, Scales, Variation, Measures of Shape (Skewness & Kurtosis).           | 11% (Suggestive) |
| **Unit 3** | **Introduction to Probability:** Random Variables, PDF/CDF, Distributions (Binomial, Poisson, uniform, etc). | 15% (Suggestive) |
| **Unit 4** | **Sampling & Estimation:** CLT, Point Estimate, Confidence Intervals, Maximum Likelihood Estimation.         | 18% (Suggestive) |
| **Unit 5** | **Simple Linear Regression:** Least Squares, Fitted Model, Inferences, Correlation.                          | 9% (Suggestive)  |
| **Unit 6** | **Logistic Regression:** Binary classification, Sensitivity, Specificity, Optimal Cut-Off.                   | 11% (Suggestive) |

## 🛠️ Practical & Case Study Topics

Key areas for hands-on learning from the syllabus:

1. Exploring traits of Big Data and Web Analytics.
2. Visualizing distributions (Binomial, Poisson) using statistical tools.
3. Calculating skewness and kurtosis for descriptive analytics.
4. Performing Simple Linear Regression on real-world datasets.
5. Implementing Binary Logistic Regression for credit rating or healthcare.
6. Building Decision Trees using ID3 and evaluating performance.
7. Naïve Bayes classification for categorical data.
8. Case Study: Data Science for driving growth in E-Commerce.
9. Case Study: Delving into customer personality analysis.

### 💡 Pro-Tips for High Marks:

> **Focus on Calculations!** Statistical distributions and regression coefficients are high-scoring calculation-based areas.

- 📌 **Distributions:** Memorize the formulas and properties for PDF/CDF of normal and student-t distributions.
- 📌 **Regression:** Always check the sensitivity and specificity values in Logistic Regression Diagnostics.
- 📌 **Decision Trees:** Understand the Gini index and Information Gain logic behind tree splitting.
- 📌 **Case Studies:** Be ready to explain the analytics capability building roadmap in exams.
